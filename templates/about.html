About QuizVerse

A student-built project designed to make learning interactive, fun, and challenging.

Our Project

QuizVerse is an interactive quiz web application created as a Computer Science project. Our goal was simple – to design a platform where users can test their knowledge, explore various topics, and enjoy an engaging quiz experience powered by clean UI and responsive design. Built using HTML, CSS, JavaScript, Python and MYSQL (Flask / MYSQL backend). it showcases how web technologies can blend fun with learning.
